# GitHub Data Dropzone

Use this folder as a temporary GitHub-backed storage area for your trading data.

## How to upload data here

1. Copy files (CSV or Parquet) into `github_data/dropzone/`.
2. Commit and push to GitHub.
3. I can then pull and use those files to power dashboard features.

## Notes

- Keep sensitive/private data out of public repos.
- Large files may need Git LFS.
